# Checklist de validación del reporte ReAS

Usar esta lista antes de dar por válido un reporte generado por el Sistema ReAS.

## Antes de procesar

- [ ] El usuario inició sesión con su código autorizado.
- [ ] El Excel principal corresponde al período que se desea evaluar.
- [ ] El archivo principal contiene las columnas requeridas.
- [ ] Se cargó el archivo de horario extendido si aplica.
- [ ] Se cargó el archivo de nómina si aplica.
- [ ] Se seleccionó el mes correcto.
- [ ] El mapeo de columnas fue revisado.

## Durante el procesamiento

- [ ] El sistema procesó sin errores visibles.
- [ ] No quedaron advertencias críticas sin revisar.
- [ ] El dashboard muestra registros procesados.
- [ ] Los totales generales parecen razonables para el período.

## Revisión de resultados

- [ ] Se revisaron ausencias.
- [ ] Se revisaron tardanzas.
- [ ] Se revisaron salidas tempranas.
- [ ] Se revisaron vacaciones.
- [ ] Se revisaron licencias.
- [ ] Se revisaron permisos.
- [ ] Se revisaron ponches irregulares.
- [ ] Se revisaron empleados con horario extendido.
- [ ] Se revisaron exclusiones por nómina.

## Auditoría

- [ ] Se revisó la auditoría de descuadres.
- [ ] Se identificaron empleados con diferencias.
- [ ] Se revisaron los registros que causan descuadre.
- [ ] Se aplicaron ajustes manuales cuando correspondía.
- [ ] El total general fue revisado.

## Excel final

- [ ] La hoja `Control del reporte` contiene el mes correcto.
- [ ] La hoja `Control del reporte` contiene el archivo original.
- [ ] La hoja `Control del reporte` contiene la hoja usada.
- [ ] La hoja `Control del reporte` contiene el usuario generador.
- [ ] Las tablas 1 a 8 están presentes.
- [ ] Las tablas son editables.
- [ ] La estructura del libro no permite mover, ocultar o borrar hojas.
- [ ] Los resúmenes están presentes.
- [ ] El archivo fue guardado con el nombre correcto.

## Entrega

- [ ] El reporte fue revisado por el responsable.
- [ ] El archivo se compartió solo por canales autorizados.
- [ ] No se expusieron datos sensibles fuera del flujo permitido.
