# Guion de presentación del Sistema ReAS

## Apertura

Buenos días. Hoy presentamos el Sistema ReAS, una herramienta diseñada para automatizar el procesamiento de reportes de asistencia y apoyar la gestión, revisión y trazabilidad de las eventualidades del personal.

ReAS nace como respuesta a un proceso que tradicionalmente requería mucha revisión manual en Excel: filtrar registros, validar observaciones, calcular horas, identificar ausencias, tardanzas, salidas tempranas y preparar tablas institucionales.

## Problema anterior

Antes, el proceso dependía de tareas manuales:

- revisar archivos Excel con miles de registros;
- filtrar por ubicación, empleado o fecha;
- calcular tiempos de forma manual o con fórmulas;
- separar eventualidades justificadas y no justificadas;
- validar descuadres visualmente;
- preparar tablas finales con formato institucional.

Esto consumía tiempo, aumentaba el riesgo de errores y dificultaba mantener una trazabilidad clara de lo procesado.

## Solución propuesta

ReAS automatiza el flujo completo:

1. carga el Excel principal de asistencia;
2. cruza horario extendido cuando aplica;
3. cruza nómina para completar datos y excluir casos especiales;
4. detecta el mes evaluado;
5. aplica reglas de cálculo;
6. audita descuadres;
7. genera indicadores, rankings y reportes;
8. exporta un Excel final institucional.

## Demostración sugerida

1. Iniciar sesión con código autorizado.
2. Mostrar el dashboard inicial.
3. Cargar el Excel principal.
4. Cargar horario extendido y nómina, si aplican.
5. Seleccionar el mes evaluado.
6. Revisar mapeo de columnas.
7. Procesar asistencia.
8. Mostrar tarjetas, gráficos y rankings.
9. Abrir auditoría de descuadres.
10. Descargar el Excel final.
11. Mostrar la hoja `Control del reporte` y las tablas principales.

## Beneficios

- Reduce trabajo manual repetitivo.
- Estandariza cálculos y reportes.
- Mejora la trazabilidad del proceso.
- Facilita la revisión por empleado, ubicación y tipo de eventualidad.
- Ayuda a detectar casos críticos de forma más rápida.
- Genera un Excel final con estructura institucional.

## Cierre

Con ReAS, Gestión Humana cuenta con una herramienta que organiza, procesa y presenta la asistencia de manera más eficiente, manteniendo controles de auditoría y trazabilidad para apoyar la revisión humana.

El sistema no sustituye la evaluación del equipo responsable, sino que reduce la carga manual y entrega información más clara para tomar decisiones.
